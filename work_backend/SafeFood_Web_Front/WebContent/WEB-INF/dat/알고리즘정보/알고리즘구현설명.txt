맞춤 섭취 조회, 알러지 주의 리스트 기능은 로그인시에만 확인하실수 있습니다.

알고리즘 기능 추가 (6가지)
■ 전체 사용자 알러지 해당 정보 (allergy_material.jsp) 
service 의 allergyMaterialRank() 에서 회원들의 전체 알러지에 대한 정보를 나누어 줌.

■ 회원정보 찾기 (findid.jsp)
회원 정보를 잃어 버렸을 경우 사용자의 등록된 이름 + 전화번호를 사용해 사용자의 정보를 찾는 기능.
아이디와 임시비밀번호를 발급해주는 알고리즘으로 사용자가 임시로 로그인이 가능.

■ 맞춤 섭취 조회 (fitcombsearch.jsp)
로그인 했을 경우에만 접근이 가능하며 
사용자가 원하는 칼로리를 입력시 그 칼로리에 95%까지 해당되는 칼로리와
사용자의 알러지 정보를 고려해서
조합알고리즘을 사용해 여러 제품의 조합을 추천.

■ 성분 첨가 빈도 (materialusechart.jsp)
전체 제품에 들어있는 모든 성분의 함유 빈도를 나타내주는 알고리즘.
쉽게 어떤 성분이 가장 많이 함유되어 있는지를 파악 가능.

■ 알러지 주의 리스트 (allergyfood.jsp)
알러지와 제품의 해당사항을 체크 해주는 알고리즘.
사용자의 해당되는 알러지와 그 알러지에 속하는 제품들을 나타내 줌.

■ 제품상세정보 페이지 알러지 알림 기능
 사용자에게 해당되는 제품은 상세페이지 에서 알러지 주의를 나타내주는 알고리즘 기능 구현.
