전체적으로 3가지 (Food, Member, Board)-식품, 회원, 게시판 기능으로 구현을 하였습니다.
각각 MVC구조로 코딩 하였으며 해당 제품에 대한 내용을 만족할 수 있도록 MVC아키텍처 기반으로 클래스를 설계 하였습니다.

각각 BoardServlet, FoodServlet, FrontServlet 이렇게 3가지로 서블렛이 나뉘며
세션과 한글처리를 위해 로그인필터와 한국어필터를 처리 하였습니다.

DB연동은 ConnectionPorxy로 Tomcat 8.5v을 사용하였습니다.
Mysql 관련 구문은 Eclipse자료에 포함되어 있습니다.

■ 메인화면(index.jsp)
- 제품 XML Parsing 후 화면에 출력 데이터 베이스와 연동 완료.
- 각 제품의 조회수에 따른 우선순위(랭크)기능 구현
- hover시 제품명 및 상품 상세페이지로 이동 버튼 팝업.
- 전화면 login Session처리 구현

■ 공지사항 리스트(notice_list.jsp)
- 공지 리스트 : 공지사항 게시판의 글쓰기, 수정, 삭제기능 구현

■ 공지사항 수정(notice_update.jsp)
-공지 수정시 수정 글에대한 정보를 DB에서 꺼내오는 기능 구현 
- 로그인된 세션과 일치할 경우에만 수정이 가능하도록 구현

■ 공지사항 글쓰기(notice_insert.jsp)
- 공지 글은 로그인 상태에서만 가능하도록 구현
- 기본적인 공지사항의 전체 리스트를 보여주는 기능 구현

■ 공지사항 리스트(notice_view.jsp)
- 공지사항 리스트의 상세 정보 표시기능 구현

■ 상품 정보(productinfo.jsp)
- 전 제품 XML Parsing 후 사진, 이름, 원재료 표시
- 제품 사진 hover시 제품명 및 제조사 slide fade 형태로 팝업. (jquery 구현)
- 조회수에 따른 우선순위 기능 구현
- 제품 검색기능 구현

■ 상품 상세 정보(productdetail.jsp)
- 파라미터로 받아온 제품의 이름으로 XML 데이터 중 해당 제품을 표시.
- 해당 제품에 대한 정보를 DB저장 및 연동.
- 해당 제품의 사진, 이름, 원재료, 제조사 표시.
- 해당 제품의 영양 정보 XML 정보를 얻어와 테이블과 chart.js로 원형그래프 표현.

■ 베스트 섭취 정보(bestintakeinfo.jsp)
- 전 제품의 상세정보와 함께 우선순위 표시.
- 각 제품의 조회수에 따른 우선순위(랭크)기능 구현

■ 회원가입(SignUp.jsp)
- 회원가입 기능 구현
- 회원가입시 알레르기 성분 설정 기능 구현.

■ Login(Login.jsp)-menu.jsp
- Login 버튼 클릭시 표현.
- 아이디 찾기, 회원정보 수정, 회원정보 삭제 구현
■(로그인과 로그아웃시의 기능 구분)
■ Logout
- Login이 되있을경우 menu nav바가 로그인한 상태의 환경으로 변경되며
  로그 아웃시 세션이 종료.
- 아이디 찾기, 회원정보 수정, 회원정보 삭제 구현

■ 회원정보 수정(updatepersonalinfo.jsp)
- 회원정보 수정 : 회원에 대한 정보를 가져오는 DB 구현. 
		그 정보를 수정하는 기능 구현
회원정보 수정시 알레르기 수정 기능 구현.

■ 회원정보 삭제(deleteinfo.jsp)
- 회원정보 삭제 구현
